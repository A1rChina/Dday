# Codex 数据库修改禁令

## 唯一权威依据

Dday V1 数据库结构唯一权威依据：

- `docs/DATABASE_LOCK_V1.md`

## Codex 禁止事项

- 不允许自行新增表。
- 不允许自行删除表。
- 不允许自行修改字段。
- 不允许自行修改状态枚举。
- 不允许自行修改 `src/db/schema.ts`。
- 不允许自行新增 migration。
- 不允许恢复 deprecated 表作为新功能事实源。

## Codex 允许事项

Codex 可以在不改数据库的前提下修改：

- services
- routes
- admin UI
- 表单映射
- 查询逻辑
- 状态流转逻辑
- 业务校验逻辑

但必须符合 `DATABASE_LOCK_V1.md`。
